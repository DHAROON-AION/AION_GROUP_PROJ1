# AION Bank — Online and Mobile Banking

## Registration
Customers register using account number, national ID, and registered mobile number. OTP verification is required.

## Security Features
- Two-factor authentication for all transactions
- Biometric login on supported devices
- Session timeout after 5 minutes of inactivity
- Transaction alerts via SMS and push notification

## Transfer Limits
- Internal transfers: up to SAR 100,000 per day
- Local SARIE transfers: up to SAR 50,000 per day (increase available on request)
- International transfers: up to SAR 25,000 per day

## Password Reset
Password reset requires OTP to registered mobile and security questions. Branch visit required if security questions fail.

## Support Hours
Digital banking support: 24/7. Complex issues escalated to branch during business hours (Sun-Thu 8:30-16:30).
